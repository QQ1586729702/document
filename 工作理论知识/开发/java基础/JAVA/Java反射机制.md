Java反射机制的理解

---

![[clipboard (14).png]]

**Class对象**的由来是将`.class`文件读入**内存**，并为之创建一个**Class对象**。

![[clipboard (15).png]]

这个**Class对象**可以看到我们想要**类的结构**，这种机制就被成为**反射**


![[clipboard (17).png]]
这个Class对象可以获取很多信息，并且可以**构造对象**，**动态代理**

![[clipboard (18).png]]
- 反射的**优势**：**动态创建和编译对象**
- 反射的**劣势**：**操作速度慢**
